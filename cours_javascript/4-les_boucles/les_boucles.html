Les boucles

Notions clés ou objectifs :

  • Les formes while..., do...while, for...
  • Utilisation des boucles
  • L'incrémentation et la décrémentation


Les boucles prennent souvent la forme d'une inégalité sur une variable. Par exemple exécuter un bout de code tant qu'une variable contient une valeur <, >, <=, >= à un nombre. Il sera donc question de faire varier la valeur contenue dans la variable à l'intérieur de la boucle, ceci à l'aide de l'incrémentation ou de la décrémetation.
Voir le resultat de l'exemple ici.

L'incrémentation ou la décrémentation peut se faire avant ou après la variable. Lorsque la variable est suivie par les signes (++ ou --) l'affectation est prioritaire sur l'incrémentation ou la décrémentation, donc la variable est d'abord affichée avant d'être augmentée ou diminuée. Si les signes sont plutôt suivis par la variable, l'augmentation ou la diminution est faite avant l'affichage de la variable.
var A = 0 , B = 0 , C = 10 , D = 10 ;
var W = A++ ;
var X = ++B ;
var Y = C-- ;
var Z = --D ;
