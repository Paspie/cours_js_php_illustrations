Notions clés ou objectifs :

  • les méthodes setInterval() clearInterval() et setTimeOut()


1- la méthode window setInterval()

Elle permet d'exécuter un script en boucle en précisant un interval de temps entre chaque repétition. Elle prend en arguments le bloc de code à exécuter et l'interval de temps en milisecondes.

3- le timeOut()

Elle permet de programmer l'exécution d'un code et prend comme arguments la fonction à exécuter et le nombre de milisecondes après lequel la fonction sera exécutée.

On utiliser le clearTimeout() pour arrêter le timeout, ceci n'est pas trop usuel.
