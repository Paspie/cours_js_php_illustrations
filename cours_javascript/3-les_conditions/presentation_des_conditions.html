Présentation des conditions en Js

Notions clés ou objectifs :

  • Définition des conditions
  • Connaître les différents opérateurs de comparaison
  • Comprendre comment le Js compare les valeurs


Les opérateurs :

"==" signifie c'est égal (en valeur)
"!=" signifie c'est différent
"===" signifie c'est égal (en valeur et en type)
"!==" signifie c'est différent (en valeur et en type)
"<" signifie c'est strictement inférieur
"<=" signifie c'est inférieur ou égal
">" signifie c'est strictement supérieur
">=" signifie c'est supérieur ou égal

Application :

var x=15 , y=26 ;
var a= x < y ;
var b = x > y ;
var c = 17 <=24 ;
var d = 17 >=24 ;
var e = 21=="21" ;
var f = 21==="21" ;
var g = 21!="21";
var h = 21!=="21" ;
