La concaténation

Notions clés ou objectifs :

  • Comprendre la concaténation
  • Pouvoir concatener avec l'opérateur "+"
  • concatener un chiffre ou une chaîne de caractères


Application :

Concatener signifie mettre bout à bout 2 chaînes de caractères afin d'en obtenir une 3ème.
Exemple :

var prenom1 = "Pascal" , espace=" " , prenom2 = "Pierre" ;
var moi = prenom1+espace+prenom2 ;
