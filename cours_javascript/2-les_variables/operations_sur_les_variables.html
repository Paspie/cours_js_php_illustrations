Opérations sur les variables

Notions clés ou objectifs :

  • Manipulation des variables
  • Opérations mathématiques simples et priorités des calculs
  • Notations condensées pour les opérations


Application

En Js, les calculs sont effectués en respectant les principes de la mathématique réelle. La priorité est donnée aux multiplications, divisions et parenthèses. les opérations peuvent s'écrivent de différentes manières, cela ne modifiera pas le résultat de l'opération. La valeur alertée correspond à la dernière valeur prise par une variable si elle a été ajoutée ou diminuée dans les opérations suivant sa déclaration.
Illustration :
var x=5 , y=4 , z=3 ;
x=x+2 ; // ou x+=2
x=x-1 ; // ou x-=1
y=y*3 ; // ou y*=3
var mult=x*y ; // pour y=y*3 par exemple, on peut aussi écrire y*=3
var divid=y/z ; // pout y=y/x , on peut écrire y*=x
var modul=17%3 // pour y=y%3 pour affecter au nouveau y le reste après sa division par 3, on peut écrire y%=3
var priorit= [6+(9+9/3)/3]%3 ;
