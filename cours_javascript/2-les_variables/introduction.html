Introduction aux variables en Js

Notions clés :

  • définition d'une variable
  • déclaration d'une variable
  • introduction aux types de valeurs
  • l'opérateur d'affection : =


Théorie :

Une variable Js est un conteneur qui va stocker temporairement une information (un nom, une chaîne de caractères, un chiffre, etc). Une variable doit pouvoir stocker différentes valeurs dans le temps en écrasant celle(s) précédente(s).
Une variable Js doit avoir un nom unique (ID), qui doit commencer par une lettre et ne peut contenir que des lettres non accentuées, peut avoir des chiffres ou un underscore ou encore un dollar. Le nom des variables Js est sensible à la casse. Il existe des mots reservés qui ne pourront pas être utilisés comme noms de variables.

Les types de valeur des variables

Une variable peut stocker un texte/chaîne de caractères(string) avec les ' ' ou " "; un chiffre ou un booléen (true/false).
Le signe = est un opérateur d'affectation et non d'égalité. Dans ce sens var X=10 signifie qu'on affecte la valeur 10 à la variable X, ou qu'on stocke 10 dans la variable X; X=10+5 , on stocke 10+5 dans X.
