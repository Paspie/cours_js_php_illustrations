l'objet date et ses méthodes

Notions clés ou objectifs :

  • créer un objet de type date
  • récupérer la date actuelle en Js
  • maîtriser les formats de dates valides
  • maîtriser les méthodes de l'objet date


L'objet date est un objet natif de Js. Il peut être écrit soit par un string, soit sous forme de time-stamp Unix, et est décompté en milisecondes.
Pour récupérer la date sous forme littérale : var date = Date() et sous forme de milisecondes : var date = Date.now() . Illustration

On peut être amené à créer ou à spécifier une date, on utilisera pour cela le constructeur date :

  • la date littérale : date()
  • spécifier une date avec un format valide : var x = new date("April 16, 2021 11:37:00")
  • En précisant un nombre en argument : var x = new date(0) (pour le time-stamp, temps écoulé depuis le 1/1/1970 en secondes) et var x = new date(604800000) (time-stamp + 1 semaine)
  • En précisant 7 nombres, les 3 premiers étant obligatoires : var x = new date(année,mois,jour,heure,minute,seconde,ms) exemple : var x = new date(2021,4,16,11,48,39,80)

Les formats de date

  • new date("2021-04-16")
  • new date("2021-04-16 T11:52:00")
  • new date("2021/04/16")
  • new date("January 25 2021 11:54:10")
  • new date("Fri Jan 16 2021 11:55:45 GMT+02:00")

Les méthodes de l'objet date()

1. Méthodes de type get-eur

Elles sont utilisées pour récupérer la date.

  • var d = new date() : on crée d'abord la date, puis on extrait les détails par les méthodes
  • var dms = d.getTime() : on récupère la date en ms
  • var dy = d.getFullYear() : on récupère l'année
  • var dmo = d.getMonth()+1 : on récupère le mois. (+1 car les mois vont de 0 à 11)

  • var dday = d.getDay() :

  • var dmi = d.getMinutes() :
  • var ds = d.getSecondes() :
  • var dMs = d.getMillisecondes() :

2. Méthodes de type set-eur

Elles sont utilisées pour définir une date. On crée donc d'abord l'objet date puis on lui attribue des propriétés

  • var d = new Date()
  • d.setFullYear(2021,3,16) (pour le 16 Avril 2021)
  • d.setHours(12)
  • d.setMinutes(22)
  • d.setSeconds(42)

Pour formater une date autrement qu'en anglais, il suffit d'appliquer ces ppt/mth et à les alerter selon l'ordre souhaité.
