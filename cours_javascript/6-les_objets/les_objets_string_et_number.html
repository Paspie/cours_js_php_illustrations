Les méthodes usuelles des objets STRING et NUMBER en Js

Notions clés ou objectifs :

  • maîtriser les méthodes usuelles des objets de type String
  • maîtriser les méthodes usuelles des objets de type Number


Propriétés et méthodes usuelles des objets de type string

Les objets de type string possèdent 3 Propriétés et une vingtaine de méthodes. On préferera utiliser les valeurs primitives string qu'aux objets string vu que les ppt/mth s'appliquent de la même façon aux 2. Les 3 pptés sont :

  • length : nomprimitive.length , en générale si on tente d'affecter une valeur à la ppt length rien d'observable ne se produira. Donc on peut utiliser cette ppt pour déterminer la longueur d'une ch car.
  • prototype : pour ajouter de nouvelles ppt/mth à un objet. En pratique pas utilisée.
  • contructor: pour retourner la fonction qui a créé le prototype. En pratique pas trop utilisée.

Les 2 dernières n'étant pas trop utilisées en pratique, nous allons nous attarder sur les méthodes de la ppt length. Ces méthodes sont sensibles à la casse.

1. les méthodes toLowerCase() et toUpperCase()

Ces méthodes ne nécessitent pas de paramètres : nomprim.toLowerCase() et nomprim.toUpperCase() . On peut évidemment faire stocker les résultats de ces mth dans d'autres variables.

2. la méthode charAt()

Elle permet de déterminer le caractère occupé par un rang dans une ch car. Elle prend en paramètre le rang en question : nomprim.charAt(rang)

3. les méthodes indexOf() et lastIndexOf()

Elles permettent de déterminer respectivement le rang de la première et la dernière occurence et d'un caract ou d'une ch car : nomprim.indexOf("car/chcar") et nomprim.lastIndexOf("car/chcar") . Elles renvoient comme résultat -1 si le car/chcar n'est pas retrouvé. On peut ajouter comme 2e paramètre facultatif la position à laquelle on souhaite démarrer la recherche.

4. la méthode remplace()

Elle permet de remplacer un car/chcar par un autre : nomprim.remplace("car/chcar à remplacer","nouveau car/chcar")

5. la méthode slice()

Elle permet d'extraire des caractères dans une chcar : nomprim.slice(rang début, rang fin) . En donnant des argument négatifs, l'extraction se fera dans le sens inverse.

6. la méthode trim()

Ne prenant pas d'argument, elle permet de nettoyer les espaces superflux ou inutiles dans une chcar : nomprim.trim()


Propriétés et méthodes de l'objet Number

On étudiera les ppt/mth des valeurs primitives de type Number et pas l'objet Number en question.
L'objet Number possède 5 propriétés qui lui apparatiennent exclusivement :

  • MIN_VALUE et MAX_VALUE : nomprimN.MIN_VALUE et nomprimN.MAX_VALUE
  • NEGATIVE_INFINITY et POSITIVE_INFINITY : nomprimN.NEGATIVE_INFINITY et nomprimN.POSITIVE_INFINITY
  • NaN : nomprimN.NaN

L'objet Number possède aussi 5 méthodes :

1. la méthode toString()

Pour transformer un type number en type string nomprimN.toString()

2. la méthode toFixed()

Pour arrondir et transformer un number float en type string. nomprimN.toFixed(nombre après la virgule)

3. la méthode toPrecision

Elle permet de choisir la taille du nombre qui sera retourné(la quantité de chiffre à renvoyer pour le nombre) et transforme le nombre en type string : nomprimN.toPrecision(qté)

4. la méthode toExponential()

Elle permet de choisir le nombre de décimaux d'un nombre et le transforme en string : nomprimN.toExponential(qté)


Quelques méthodes globales de Js

Ces méthodes peuvent être utilisées avec tous les types de données.

1. la méthode Number()

Elle permet de convertir certaines variables Js en nombre comme des boléens ou des dates : nomprimN.Number(boléen/date)

2. les méthodes parseInt() et parseFloat()

Elle permettent de retourner un nombre entier ou décimal à partir d'un nombre de type string : parseInt(nomprimN) et parseFloat(nomprimN)
