les fonctions en javascript

Notions clés ou objectifs :

  • Les fonctions :utilisation et syntaxe
  • création d'une fonction et utilisation des paramètres/arguments


Une fonction est un bloc de code qui effectuera une tâche précise, par exemple la fonction "alert" qui permet d'afficher des messages dans la boîte de dialogue.
L'intérêt des fonctions est qu'on a pas besoin de redéfinir leurs codes à chaque fois; une fois créée, il se suffira de l'appeler pour qu'elle s'exécute à un endroit précis.
Il existe 2 grands types de fonction en Js :

  • les fonctions Natives : qui sont des fonctions déjà prêtes à l'emploi;
  • les fonctions créées par nous-mêmes : pour nos besoins spécifiques

Nous allons ici analyser le 2e type de fonction.

La mise en marche d'une fonction se fait en 2 étapes : la création et l'appel de la fonction. De même que la fonction "alert" est souvent utilisée dans d'autres fonction pour afficher un message, une fonction créée par nous mêmes peut être utilisée à l'intérieur une autre.

Les fonctions n'ont pas forcément besoin des paramètres pour fonctionnner. On peut se poser la question de quoi a besoin notre fonction pour fonctionner ? pour lui trouver ou pas des paramètres; par contre les fonctions ont toujours besoin des ( ) même si elles sont vides. Les paramètres deviennent des arguments lorsqu'ils sont définis dans l'appel de la fonction.

L'instruction return() fera récupérer le résultat d'une fonction sans l'afficher tandis que alert() l'affichera immédiatement. Il est recommandé de plus souvent utiliser return() , avec elle on peut capturer le résultat d'une fonction généralement en utilisant une autre variable afin de pouvoir le réutiliser. On finira toujours par utiliser alert() pour afficher le resultat. A noter que l'instruction return() met fin à l'exécution d'une fonction, le reste du code après elle ne sera simplement pas exécuté.
