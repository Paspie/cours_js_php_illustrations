  • Les dégradés : linéaires, radiaux

Le Canvas est un élément html qui permet de représenter des figures géométriques au sein d'une page web. Il ne sert que de conteneur pour les graphiques, ceux étant construits par des langages de script tel que le Js. L'élément Canvas représente une zone rectangulaire n'ayant par défaut ni bordure ni contenu. On déclare le Canvas (lui donnant une longueur&largeur) au sein de la page, on le stylise en CSS et on le récupère avec le DOM afin de faire les figures.
