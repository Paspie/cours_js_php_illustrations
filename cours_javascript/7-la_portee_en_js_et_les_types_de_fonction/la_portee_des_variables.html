La portée des variables en Js

Notions clés ou objectifs :

  • Savoir où une variable sera accessible ou pas


Une variable globale est une variable définie dans l'espace global du script, autrement dit à l'extérieur de toutes fonctions, et sera donc accessible à n'importe quel endroit du script même à l'intérieur d'une fonction tandis qu'une variable locale n'est accessible qu'à l'intérieur de la fonction où elle a été définie.Il n'est pas recommandé d'utiliser les variables globales dans des espaces locaux, et est déconseillé d'utiliser le même nom pour plusieurs variables.
Lorsqu'une variable est définie dans un espace local sans la mention "var", elle se comporte comme une variable globale. Il est bien de ne déclarer des variables globales que si c'est nécessaire.
