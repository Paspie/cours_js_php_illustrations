Partie introductive

Ce cours de javascript est constitué des notions les plus usuelles dans ce langage, et est illustré par des exemples les plus communs. Le javascript est un langage de programmation majoritairement utilisé client-side, orienté objet, qui va permettre de manipuler dynamiquement le html et le css des pages.
Le code Js peut être écrit aussi bien dans la partie "head" ou "body" de la page html que dans un autre fichier séparé (voir les commentaires relatifs aux script js de ce document).
Il existe aussi ici des monocommentaires (//) et des multicommentaires (/*...*/), l'illustration a bien été faite de le script de ce cours.
